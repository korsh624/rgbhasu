Проблема

Робот-кибер-помощник машиниста обходчика. Работа на ТЭЦ – это высокая степень ответственности и внимательности каждого сотрудника, а также непрерывное обучение персонала. Это занимает много ресурсов и времени, а сотрудник без должной подготовки может получить травмы или привести оборудование в негодность

Наше решение многоагентная система сети роботов.

Каждый робот работает на своем этаже и ему не приходится преодолевать лестничные марши

  • 01 Навигация робота

    Навигация осуществляется с помощью специальной разметки на полу, робот движется при помощи камеры

  • 02 Определить где какой датчик?

    Датчики определяются с помощью NFC меток расположенных у каждого датчика, в манипуляторе вмонтирован ридер.

  • 03 Чтение с аналоговых датчиков ?

    Чтение с аналоговых датчиков производится с помощью компьютерного зрения, библиотерка OpenCV

 

Навигация

Робот осуществляет навигацию по своему этажу с помощью цветной разметки нанесенной на пол. С помощью сегментации по цвету эта разметка легко определяется.

Определение датчиков

Робот распознает датчики с помощью NFC меток. Считыватель встроен в манипулятор.

Чтение с аналоговых датчиков

Чтение с аналоговых датчиков производится с помощью компьютерного зрения, реализованного на языке высокого уровня Python, библиотерка OpenCV.
